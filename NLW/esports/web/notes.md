# Instalação tailwind CSS

```
npm install  -D tailwindcss postcss autoprefixer
```

## Iniciar tailwind
```
npx tailwindcss init -p
```

## Configurar arquivo tailwind.config
```
content: [
    './src/**/*.tsx,*.jsx'
  ],
```

## Criar arquivo styles/main.css
```
@tailwind base;
@tailwind components;
@tailwind utilities;

```
## Instalar extensoes
postcss
tailwind

npm install phosphor-react
